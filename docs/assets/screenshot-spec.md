## 六、截图执行建议

1. 本地启动江南出行 (Docker Compose up)
2. 微信开发者工具打开两地书项目
3. 浏览器打开题库 HTML 文件
4. 使用 Chrome DevTools 截取全页面 (Device Mode: 1440x900)
5. 裁剪到内容区域，保存为 PNG (2x 分辨率)
6. 文件命名: `{project}-{page-name}.png` 放入 `docs/assets/screenshots/`
