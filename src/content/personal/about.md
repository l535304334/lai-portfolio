## 工程定位

软件工程专业大三学生。不是"会用 Spring Boot 的学生"，而是一个能独立分析并发调度问题、设计分布式锁方案、写形式化状态机、并用压力测试验证的工程实践者。

三个项目，从单文件 HTML 到企业级全栈系统，从需求分析到部署上线——每个项目都是一次完整的工程实践闭环。

## 成长轨迹

从单文件 HTML 到企业级全栈系统，三个项目构成一条完整的能力跃迁路径。每个项目都建立在前一个的局限之上，推动我从客户端工程化走向服务端架构，再到分布式系统。

## 关于本站

本网站使用 Vue 3 + TypeScript + Vite 构建。所有项目内容以 Markdown 编写，构建时转为静态 HTML。不使用 UI 框架——所有组件手写，追求"内容优先、克制专业"的 Developer Academic 风格。

设计理念：不是告诉面试官我有多强，而是展示我怎么解决问题。
